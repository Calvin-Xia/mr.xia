# Mr.Xia - 个人网站

> 简约现代风格的个人网站，展示作品与创意

## ?? 项目简介

这是一个采用现代化设计风格的个人网站项目，具有以下特点：

- ?? **简约美观**：采用渐变色彩和微交互设计
- ?? **响应式布局**：完美适配各种设备
- ? **性能优化**：轻量级代码，快速加载
- ??? **实用工具**：内置在线计时器等工具
- ? **动画效果**：流畅的页面过渡和交互动画

## ??? 项目结构

```
mr.xia/
├── css/
│   └── style.css          # 统一样式表
├── js/
│   └── main.js            # 主脚本文件
├── index.html             # 首页
├── Works.html             # 作品展示页
├── timetable.html         # 计时器工具页
├── statement.html         # 网站声明页
├── 404.html               # 404错误页
└── README.md              # 项目说明
```

## ?? 设计亮点

1. **渐变动画背景** - 柔和的渐变色彩动态变化
2. **毛玻璃导航栏** - 现代化的半透明效果
3. **悬浮卡片动效** - 鼠标悬停时的流畅动画
4. **微交互按钮** - 波纹扩散效果
5. **页面进入动画** - 元素依次淡入呈现

## ?? 快速开始

1. 克隆项目到本地
2. 使用任意 Web 服务器打开 `index.html`
3. 或直接双击 `index.html` 在浏览器中打开

## ?? 技术栈

- HTML5 - 语义化标签
- CSS3 - 现代化样式（CSS变量、渐变、动画）
- JavaScript - ES6+ 模块化代码
