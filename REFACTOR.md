# ?? 项目重构说明

## ?? 重构目标

本次重构旨在将原有的个人网站项目进行全面优化，实现：
- ? **统一代码风格** - 规范化所有代码
- ? **现代化设计** - 简约但有亮点的视觉效果
- ? **优化结构** - 清晰的目录组织
- ? **提升体验** - 流畅的动画和交互

## ?? 新的目录结构

```
mr.xia/
├── css/                    # 样式文件目录
│   └── style.css          # 统一样式表（包含所有页面样式）
├── js/                     # 脚本文件目录
│   └── main.js            # 主脚本（模块化设计）
├── index.html             # 首页
├── Works.html             # 作品展示页
├── timetable.html         # 工具页（计时器）
├── statement.html         # 声明页
├── 404.html               # 404错误页
├── project.json           # 项目配置
├── .gitignore             # Git忽略文件
└── README.md              # 项目说明文档
```

## ?? 设计亮点

### 1. 渐变动画背景
- 使用CSS渐变和关键帧动画
- 柔和的色彩过渡效果
- 低透明度不影响内容阅读

### 2. 毛玻璃导航栏
- backdrop-filter 实现模糊效果
- 半透明背景提升现代感
- 粘性定位始终可见

### 3. 悬浮卡片动效
- transform 实现平滑上浮
- 阴影变化增强立体感
- 边框颜色渐变提示交互

### 4. 微交互按钮
- 波纹扩散效果
- 渐变背景
- 立体阴影

### 5. 页面进入动画
- 元素依次淡入
- 使用 fadeInUp 动画
- 延迟时间递增

## ?? 技术特性

### CSS 变量系统
使用 CSS 自定义属性统一管理：
- 颜色主题
- 间距规范
- 阴影层级
- 圆角大小
- 过渡时间

### 模块化 JavaScript
- TimeDisplay 模块 - 时间显示
- Timer 模块 - 计时器功能
- PageAnimations 模块 - 动画效果
- Navigation 模块 - 导航高亮
- Utils 模块 - 工具函数

### 响应式设计
- 移动端优先
- 断点：768px、480px
- 灵活的布局适配
- 触摸友好的交互
