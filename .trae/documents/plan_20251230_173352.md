# 为所有图片添加备注文本显示功能

## 实现方案

### 处理范围
同时处理两种图片引用方式：

1. **HTML格式**：`<img src="..." alt="备注文本">`
2. **Markdown格式**：`![备注文本](image.jpg)`

### 实现步骤

1. **修改statement.html中的JavaScript**
   - 在MarkdownRenderer模块中添加图片处理函数
   - 遍历渲染后的HTML中所有img元素
   - 对于Markdown语法引用的图片，marked.js会自动转换为img标签，alt属性会被保留
   - 为每个有alt属性的img创建备注文本容器

2. **添加CSS样式**
   - 创建.image-caption样式类
   - 字体大小：0.85em（小于正文1em）
   - 颜色：var(--text-secondary)
   - 居中显示，适当的间距

### 备注文本显示效果
- 位置：每张照片正下方
- 字体大小：0.85em
- 颜色：次要文本颜色
- 对齐：居中
- 上边距：4px

### 验证步骤
1. 打开statement.html页面
2. 检查HTML格式图片（成人礼2.jpg等）下方是否显示备注文本
3. 检查Markdown格式图片（足迹地图等）下方是否显示备注文本
4. 验证字体大小和颜色是否符合要求
